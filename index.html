Github uses step by step

at first you create a new reposetory on github.com

first we can export our code using command prpmpt

open command prompt and follow

  1. now to end up on the directory we either have to one step up or down or in simple language have to go one step forward or backward.
  2. to go backward type:cd.. and press tab.
  3. then to go forward type: cd_name of the directory.
  4. at last when we end up on the correct directory.... type:git_--version and press tab to check if git is respondinga or not.
  5. then paste first command line:echo "# my-thrid-repo" >> README.md.
  6. one thing to remeber is that:- 1st command line is not mendatory. we won't have use that from the second repository
  7. copy 2nd command line:git init
  8. copy 3rd command line:git add README.md
  9. NOTE:on the 3rd command line we need to use (.) instead of README.md
    because using (.) will add everything
